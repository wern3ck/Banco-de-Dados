# 🗄️ Banco de Dados

Repositório dedicado aos estudos e atividades práticas da disciplina de **Banco de Dados**, cursada no UniCEUB — Ciência da Computação.

---

## 📁 Estrutura do Repositório

| Arquivo / Pasta | Descrição |
|---|---|
| `primeiros comandos SQL` | Introdução à linguagem SQL com os primeiros comandos DDL e DML |
| `Atividade agregação e filtragem` | Exercícios com funções de agregação (`COUNT`, `SUM`, `AVG`) e cláusulas de filtragem (`WHERE`, `HAVING`) |
| `MySQL` | Scripts e configurações utilizando o MySQL |
| `MER.png` | Modelo Entidade-Relacionamento (diagrama conceitual) |
| `modelo lógico.png` | Modelo Lógico do banco de dados |
| `modelo físico.png` | Modelo Físico com estrutura de tabelas e tipos de dados |

## 📚 Conteúdos Abordados

- **Modelagem de dados** — MER, Modelo Lógico e Modelo Físico
- **DDL** — `CREATE`, `ALTER`, `DROP`
- **DML** — `INSERT`, `UPDATE`, `DELETE`, `SELECT`
- **Filtragem** — `WHERE`, `BETWEEN`, `LIKE`, `IN`
- **Agregação** — `GROUP BY`, `HAVING`, `COUNT`, `SUM`, `AVG`, `MIN`, `MAX`
- **Joins** — relacionamento entre tabelas
